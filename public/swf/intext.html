Here you can see an example of embedding the Carousel in a common html page. The flash object is 750x500 pixels and is included via SWFObject inside a container div. The container div has CSS styles to set borders and margins.

To add images edit the xml file provided with this example. There are many options that you can set via xml file, refer to www.flshow.net for a complete list.

This slideshow requires Adobe Flash Player 9.0 (or higher). JavaScript must be enabled.
